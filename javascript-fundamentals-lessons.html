Variables: The Containers of Programming

Variables are fundamental to programming. They act as containers for storing data values, allowing you to store, retrieve, and manipulate information throughout your code.

Variable Declaration Methods

  • var: Function-scoped, can be re-declared and updated
  • let: Block-scoped, can be updated but not re-declared
  • const: Block-scoped, cannot be updated or re-declared

Code Example


// Variable Declaration
var oldSchoolVar = "I can be re-declared";
let modernLet = "Block-scoped variable";
const unchangeableConst = "Cannot be reassigned";

// Primitive Data Types
let stringType = "Hello, World!";     // String
let numberType = 42;                  // Number
let booleanType = true;               // Boolean
let undefinedType;                    // Undefined
let nullType = null;                  // Null

// Complex Data Types
let arrayType = [1, 2, 3, 4, 5];      // Array
let objectType = {                    // Object
    name: "John",
    age: 30
};

// Type Checking
console.log(typeof stringType);       // "string"
console.log(typeof numberType);       // "number"
console.log(Array.isArray(arrayType)); // true
